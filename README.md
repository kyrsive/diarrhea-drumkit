# diarrhea-drumkit
a collection of underground rap (tdf/perc40) styled drum sounds from [diarrhea drumkit vol 2](https://www.reddit.com/r/Drumkits/comments/1hr9iai/diarrhea_drumkit_vol_2_perc40_tdf_sounds/) by prodryhon to be used on strudel

to use this inside of strudel, enter the following line in your strudel pattern:
```
samples('github:kyrsive/diarrhea-drumkit')
```

please [support ryhon](https://www.instagram.com/prodryhon/) if u can
